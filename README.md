# ***GERMAN***
Ein simples **Bodycam Overlay**, das eine realistische Bodycam-Anzeige für Polizei- und Rettungsjobs aktiviert.

Die Bodycam zeigt den **Charakternamen**, das **Datum**, die **Uhrzeit** und kann nur von berechtigten Jobs genutzt werden.

---

# 🔧 Features

- Bodycam UI
- Anzeige von Name + Datum + Uhrzeit
- Mehrere Datumsformate (EU / US / ISO)
- Item (ox_inventory kompatibel)
- Automatische Deaktivierung wenn Bodycam im Inventar fehlt
- ox_lib Progressbar

---

# 🚀 Installation

1. Lege den Ordner `CRW_bodycam` in deinen `resources/` Ordner.
2. Füge folgendes in deine `server.cfg` ein:
 `            ensure CRW_bodycam            `
3. Richte das Script in der Config so ein, wie es für dich passt.


# ***ENGLISH***
A simple **bodycam script for GTA V / FiveM** that enables a realistic bodycam display for police and emergency response jobs.

The bodycam displays the **character name**, the **date**, and the **time**, and can only be used during authorized jobs.

---

# 🔧 Features

- bodycam UI
- Displays name + date + time
- Multiple date formats (EU / US / ISO)
- Item (ox_inventory compatible)
- Automatic deactivation if bodycam is missing from inventory
- ox_lib progress bar

---

# 🚀 Installation

1. Place the `CRW_bodycam` folder in your `resources/` folder.
2. Add the following to your `server.cfg`:
 `            ensure CRW_bodycam            `
3. Configure the script in the config file to suit your needs.
