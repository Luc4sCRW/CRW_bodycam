# ***GERMAN***
Ein simples **Bodycam Script für GTA V / FiveM**, das eine realistische Bodycam-Anzeige für Polizei- und Rettungsjobs aktiviert.

Die Bodycam zeigt den **Charakternamen**, das **Datum**, die **Uhrzeit** und kann nur von berechtigten Jobs genutzt werden.

---

# 🔧 Features

- Realistische Bodycam UI
- Anzeige von Name + Datum + Uhrzeit
- Mehrere Datumsformate (EU / US / ISO)
- Sekunden optional ein-/ausschaltbar
- Job-System (Whitelist)
- Item-basiert (ox_inventory kompatibel)
- Automatische Deaktivierung wenn Item fehlt
- Sound Effekte beim Ein-/Ausschalten
- ox_lib Progressbar + Animation
